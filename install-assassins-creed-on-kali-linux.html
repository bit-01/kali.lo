Install Assassin's Creed On Kali Linux

Assassin's Creed is a very unique game that combines the action and sandbox elements to create an immersive story-line in the time period of the Crusades. Not only can you climb any structure and are free to strategize how to make your kill, Assassin's Creed also gives you the freedom to save people in distress and roam massive ancient cities.

Now you can install it and play it on kali linux using:

  • PlayOnLinux: 4.2.5
  • Wine: 1.6.2
